# XenOS
```sh
# Dependencies: git, node, npm (or) pnpm, uuidgen
$ git clone https://github.com/nebulaservices/xenos.git --recursive
$ cd xenos

$ pnpm i

$ pnpm build # Linux, MacOS, etc.
$ pnpm build:windows # Windows

$ pnpm dev # Linux, MacOS, etc.
$ pnpm dev:windows # Windows

# For production:
$ pnpm start
```
