# XenOS
```sh
# Dependencies: git, node, npm (or) pnpm, uuidgen
$ git clone https://github.com/nebulaservices/xenos.git --recursive
$ cd xenos

$ pnpm i

# UNIX-like shells (Linux, MacOS, etc): 
$ pnpm build

# Powershell (Windows):
$ pnpm build:windows

# For development:
$ pnpm dev

# For production:
$ pnpm start
```
